# 圣人工作流 · 洪荒天道

> 道生一，一生二，二生三，三生万物。
> 今奉鸿钧法旨，开启洪荒圣域，调度诸圣，共襄盛举。

## 使用场景

- 复杂功能开发（阐截对抗）
- Bug 修复与优化（女娲补天）
- 架构重构设计（通天布阵）
- 代码质量审查（西方双审）
- 测试验证修复（伏羲女娲）

## 天道监管

任务开始时自动开启天道法眼，每三刻降临监察：

```
CronCreate(cron: "*/3 * * * *", prompt: "/saints-workflow:tiandao", recurring: true)
```

功德圆满后自动退场：

```
CronDelete(job_id: {天道令牌})
```
