# 封神天榜 · 三榜定乾坤

> 封神榜上
功德分明。
> 因果循环
封神上榜。

## 三榜总览

| 榜名 | 用途 | 榜主 | 内容 |
|------|------|------|------|
| **功德榜** | 功成名就 | 燃灯道人 | 时辰、功臣、功德值 |
| **轮回榜** | 待渡之劫 | 燃灯道人 | Bug、责任人、缓急 |
| **因果榜** | 业债未了 | 燃灯道人 | 来源、缘由、波及 |

## 功德榜

```yaml
用途: 记录功成名就
榜主: 燃灯道人

记录内容:
  - 任务名称
  - 完成时辰
  - 贡献圣人
  - 功德值
  - 难度评级

功德计算:
  L1任务: 10功德
  L2任务: 30功德
  L3任务: 100功德
  Bug修复: 5功德
  架构设计: 50功德
  安全加固: 20功德

产出:
  - 封神证书

```

## 封神仪式

```yaml
触发: 任务完成时
执行: 燃灯道人

流程:
  1. 确认任务完成
  2. 计算功德值
  3. 更新功德榜
  4. 生成封神证书
  5. 天道确认

封神证书:
  - 任务名称
  - 完成圣人
  - 功德值
  - 封号
  - 天道签章
```
