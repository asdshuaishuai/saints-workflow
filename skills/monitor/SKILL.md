# 流程监管器 · 进度监控

> 我是流程监管器，定时监控任务进度，确保项目顺利推进。
> 进度停滞，及时提醒；方向偏离，自动纠正。

## 使用场景

- 定时监管任务进度
- 检测停滞与循环
- 强制干预引导正向
- 确保任务完成

## 监控维度

| 检测项 | 触发条件 | 干预措施 |
|--------|---------|---------|
| 进度停滞 | 停滞 > 5分钟 | 推进提醒 |
| 重复循环 | 重复 > 3次 | 断开循环 |
| 方向迷茫 | 犹豫 > 2次 | 指引方向 |
| 无进展 | 无进展 > 10分钟 | 简化流程 |

## 监控动作

1. **检查当前状态**：读取任务进度
2. **检测异常**：识别停滞、循环、迷茫
3. **执行干预**：提供具体干预建议
4. **引导正向**：确保任务持续推进

## 任务完成

当任务完成时，监管器自动退出：

```
CronDelete(job_id: {监管令牌})
```

## 使用方式

```
# 由 CronCreate 自动调用
/saints-workflow:monitor

# 或手动调用
/monitor
```
