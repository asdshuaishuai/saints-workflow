GitHub 仓库 Gitee 镜像

核心特性

融合现代项目管理与 AI 技术，打造高效的开发协作体验

⏰

定时监管机制

每两分钟自动检查，检测停滞、循环、偏离，自动干预引导回正轨。

🎯

动态资源分配

根据任务复杂度自动选择 opus/sonnet/haiku 模型，平衡性能与成本。

🔄

并行任务调度

复杂任务自动识别可并行执行部分，协调多个 Agent 并行工作。

🧠

智能流程优化

简单任务自动跳过不必要阶段，L1 任务直接进入开发环节。

🌍

多语言技术栈

支持 Python、JavaScript、Go、Rust 等 10+ 编程语言，WebSearch 动态学习新技术。

📊

产研对抗模式

开发团队与测试团队对抗协作，王对王、将对将，确保产品质量。

安装指南

三步完成本地安装

克隆插件到本地缓存

git clone -b feature/enterprise-version https://github.com/asdshuaishuai/saints-workflow.git \ ~/.claude/plugins/cache/local/saints-workflow/1.2.0

注册插件

编辑 ~/.claude/plugins/installed_plugins.json 添加配置
详见 README.md

启用并重启

编辑 ~/.claude/settings.json 添加到 enabledPlugins
然后运行 claude 重启
