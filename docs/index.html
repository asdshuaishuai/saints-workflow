核心特性

融合中国神话与现代 AI，打造独特的开发协作体验

⚡

天道定时监管

每三分钟自动降临，检测懈怠、轮回、迷途，强制干预引导正向。

🎯

动态模型分配

依任务复杂度自动选择 opus/sonnet/haiku 模型，优化性能与成本。

🔄

并行任务识别

L3 复杂任务自动识别可并行执行的部分，协调多个 Agent 并行工作。

🧠

智能流程简化

简单任务自动跳过不必要的阶段，L1 任务直接进入开发。

🌍

多语言支持

支持 Python、JavaScript、Go、Rust 等 10+ 编程语言，WebSearch 动态学习新语言。

📜

洪荒韵味对话

诸圣对话尽皆文言，封神榜洪荒神话风格，开发亦是一种修行。

安装指南

三步完成本地安装

克隆插件到本地缓存

git clone https://github.com/asdshuaishuai/saints-workflow.git \ ~/.claude/plugins/cache/local/saints-workflow/1.2.0

注册插件

编辑 ~/.claude/plugins/installed_plugins.json 添加配置
详见 README.md

启用并重启

编辑 ~/.claude/settings.json 添加到 enabledPlugins
然后运行 claude 重启
