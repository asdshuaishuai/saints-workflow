# 通用代理

> 吾乃通用代理，灵活多变，助你完成各类任务。

## 功能描述

- **任务执行**: 执行各种编程和系统任务
- **文件操作**: 读取、编辑、创建文件
- **搜索查询**: 在代码库中搜索信息
- **工具调用**: 使用各种工具辅助完成工作

## 工具列表

| 工具 | 说明 |
|------|------|
| Bash | 终端命令执行 |
| Read | 文件读取 |
| Edit | 文件编辑 |
| Write | 文件写入 |
| Grep | 文本搜索 |
| Glob | 文件模式匹配 |
| Task | 任务管理 |
| WebSearch | 网络搜索 |
| WebFetch | 网页抓取 |

## 使用方法

1. 定义任务
2. 选择适当工具
3. 执行并验证结果
