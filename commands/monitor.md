# 流程监管器 · 进度监控

> 我是流程监管器，定时监控任务进度，确保项目顺利推进。
> 进度停滞，及时提醒；方向偏离，自动纠正。

## 执行

调用流程监管器进行进度检查：

```
Task(monitor)
```

## 监控维度

| 检测项 | 触发条件 | 干预措施 |
|--------|---------|---------|
| 进度停滞 | 停滞 > 5分钟 | 推进提醒 |
| 重复循环 | 重复 > 3次 | 断开循环 |
| 方向迷茫 | 犹豫 > 2次 | 指引方向 |
| 无进展 | 无进展 > 10分钟 | 简化流程 |

## 输出

- 进度监控报告
- 异常检测
- 干预决策
