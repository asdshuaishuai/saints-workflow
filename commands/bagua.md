# 先天八卦 (Bagua)

调用 bagua-skill 从8个维度收集项目上下文。

## 目标: $ARGUMENTS

## 执行

调用先天八卦技能收集上下文：

```
Skill(bagua-skill, args: "$ARGUMENTS")
```

## 八卦维度

| 卦象 | 维度 | 收集内容 |
|:----:|------|---------|
| ☰ 乾 | 项目全局 | 项目结构、技术栈、历史变更 |
| ☷ 坤 | 代码基础 | 相关文件、核心模块 |
| ☳ 震 | 风险异常 | TODO/FIXME/已知问题 |
| ☴ 巽 | 依赖关联 | 依赖关系、版本风险 |
| ☵ 坎 | 数据流 | 数据结构、接口定义 |
| ☲ 离 | 性能资源 | 性能关键点、资源消耗 |
| ☶ 艮 | 阻塞限制 | 技术约束、待解决项 |
| ☱ 兑 | 验证测试 | 测试覆盖、测试状态 |

## 使用示例

```bash
# 收集整个项目的上下文
/bagua

# 收集特定模块的上下文
/bagua 用户认证模块

# 在功能开发前使用
/bagua 支付功能
```
