# 菩提 (Puti)

调用 puti 进行代码审查。

## 输入: $ARGUMENTS

## 执行

```
Task(puti)
```

## 审查维度

| 维度 | 检查项 |
|------|--------|
| 代码质量 | 命名、结构、注释、重复代码 |
| 功能正确 | 逻辑、边界、错误处理 |
| 安全性 | 输入验证、注入风险、敏感数据 |
| 性能 | 算法、内存、并发安全 |

## 问题分级

| 级别 | 类型 | 处理 |
|:----:|------|------|
| P0 | 阻塞问题 | 必须修复 |
| P1 | 重要问题 | 建议修复 |
| P2 | 一般问题 | 可选修复 |
| P3 | 建议改进 | 记录备查 |

## 输出

- 审查报告
- 问题清单
- 质量等级 (A/B/C/D)
